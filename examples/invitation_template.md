# 天賦問卷 — 給朋友的邀請文字範本

> 使用情境：你決定做天賦問卷後，把這段傳給 5–6 位**認識你**的人（家人、朋友、同事都可以），收回他們的回覆，再貼回 Claude 對話讓 AI 彙整。

---

## 預設版本（直接複製）

```
嗨！我在用 NextRole 做一個求職盤點，想請你幫我寫「你眼中的我」 —— 不用全部回答，挑你有感覺的回就好，講具體例子最有用 🙏

1. 你覺得我最突出的天賦或能力是什麼？（例如：給建議、分析、建立事物、鼓勵他人、教練/帶人、建立連結、協助貢獻、創造、設計、執行、直覺判斷、發明、領導、傾聽、規劃安排、協調、拓展人脈、化解衝突、策略思考、深度思考，或其他？）

2. 如果要向別人介紹我，你會用哪些形容詞描述我？（個性特質、工作風格、人格特質都可以）

3. 你能分享一些我曾展現上述天賦或能力的具體例子嗎？（有沒有什麼事件讓你印象特別深刻？）

4. 你什麼時候覺得我在工作中最有熱情、最投入或最有成就感？（當時我在做什麼？）

5. 你什麼時候覺得我在工作中最不快樂、最沒有動力或最挫折？（當時的情境是什麼？）

6. 如果由你來建議，你覺得我適合從事什麼工作？（你會怎麼想像我的職涯未來發展？為什麼？）

7. 你覺得我在哪些環境下表現特別好？哪些環境不適合我？
```

## 用法

1. 把上面那段傳給對方。
2. 朋友回覆陸續到了，**逐則貼回** Claude 對話：「朋友A 說：xxx」「同事B 說：yyy」。
   - 也可以一次貼全部，或彙整到 FigJam / Google Sheet 後把連結貼進來。
3. 收到 ≥ 3 則（軟標準）後，跟 Claude 說「彙整吧」，它會整理成共通天賦關鍵字。
