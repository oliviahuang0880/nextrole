# 天賦問卷 — 給朋友的邀請文字範本

> 使用情境：你決定做天賦問卷後，把這段傳給 5–6 位**認識你**的人（家人、朋友、同事都可以），收回他們的回覆，再貼回 Claude 對話讓 AI 彙整。

---

## 預設版本（直接複製）

```
嗨！我在用 NextRole 做一個求職盤點，想請你幫我寫幾句「你眼中的我」 ——

✨ 我有什麼天賦或特質？
✨ 我做什麼事的時候看起來特別有勁、特別投入？
✨ 別人遇到什麼事會想到要找我？

不用長篇大論，幾句話、講具體例子就好。越具體越好，謝謝你 🙏
```

## 用法

1. 把上面那段（從 `嗨！` 到 `🙏` 結尾）傳給對方。
2. 朋友回覆陸續到了，**逐則貼回** Claude 對話：「朋友A 說：xxx」「同事B 說：yyy」。
   - 也可以一次貼全部，例如：
     ```
     朋友A（朋友）：她超會分析，總能在一堆雜訊中找出重點⋯
     家人B（家人）：從小就很會跟人聊天，朋友很多⋯
     同事C（同事）：規劃力強，每次接專案都很有條理⋯
     ```
3. 收到 ≥ 3 則（軟標準）後，跟 Claude 說「彙整吧」，它會整理成共通天賦關鍵字。
